# Citizen Hero

## Usage

 -- This app requires bun and node 21+
 ### Install Node and NVM

 ### Install Bun
    -Linux and MacOS : `curl -fsSL https://bun.sh/install | bash`
    -MacOS : `brew install oven-sh/bun/bun`
    or follow
    https://bun.sh/docs/installation
 ### Run app
````
    bun install
    bun run
````
